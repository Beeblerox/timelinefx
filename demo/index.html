timelinefxjs demo

JavaScript port of the timelinefx particle system
with pixijs rendering

Select a particle effect from the menu at left.

The timelinefxjs Github repository (including this demo) can be found here

This demo works best in modern browsers with WebGL support. While pixijs will fallback to canvas rendering, performance will sufffer particulary for effects with many particles. The timelinefxjs module itself is independant of rendering, meaning it can be used with your own html5 renderer or gameengine.
